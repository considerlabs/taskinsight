# TaskInsight — UI/UX 화면 명세

> 모든 신규 화면은 기존 디자인 토큰(`globals.css`) 사용.
> 폰트: Pretendard Variable. 아이콘: lucide-react.
> 사이드바 너비: 72px (변경 없음).

---

## 라우팅 구조

```
/login                          ← 인증 불필요 (로그인 페이지)
/(app)/                         ← 인증 필요 레이아웃 (사이드바 포함)
  /flow                         ← 기존 유지
  /dashboard                    ← 기존 유지
  /reports/weekly               ← 기존 유지
  /settings                     ← 기존 유지
  /projects                     ← 신규: 프로젝트 목록
  /projects/[id]                ← 신규: 프로젝트 홈 (이슈 목록 기본)
  /projects/[id]/board          ← 신규: 칸반 보드
  /projects/[id]/gantt          ← 신규: Gantt 차트
  /projects/[id]/milestones     ← 신규: 마일스톤 목록
  /projects/[id]/settings       ← 신규: 프로젝트 설정 (관리자)
  /issues/[id]                  ← 신규: 이슈 상세 (전체 화면)
  /admin/users                  ← 신규: 사용자 관리 (시스템 관리자)
  /admin/system                 ← 신규: 시스템 설정
```

**레이아웃:** 중앙 정렬 카드 (사이드바 없음)


## 화면 9: 프로젝트 설정 (`/projects/[id]/settings`) — 관리자 전용

탭 구성:
1. **기본 정보** — 프로젝트명, 설명, 기본 담당자
2. **멤버 관리** — 멤버 목록, 역할 변경, 추가/제거
3. **워크플로우** — 상태 추가/수정/순서변경, 전환 규칙 설정
4. **위험** — 프로젝트 비활성화
